# Software Engineering Projects Portfolio

This repository contains a collection of small and medium-scale projects implemented for learning, experimentation, and software engineering practice.

The projects focus on:
- networking fundamentals
- automata theory
- backend logic
- validation systems
- algorithmic problem solving
- systems programming concepts

---

## Projects

### Mini HTTP Server
A minimal HTTP server implemented using Python sockets and threading.

Features:
- HTTP request parsing
- simple router
- REST API
- JSON responses
- in-memory todo storage
- multi-threaded connections

Concepts:
- socket programming
- HTTP protocol basics
- REST design
- concurrency

---

### Schema Validator
A validation engine that checks JSON data against schema constraints and produces structured error reports.

Features:
- schema validation
- constraint checking
- structured error reporting
- Docker execution
- test validation workflow

Concepts:
- defensive programming
- data validation
- containerized execution
- JSON processing

---

### Pushdown Automaton (PDA) Simulator
A simulator for a pushdown automaton capable of processing input strings using stack-based computation.

Features:
- stack operations
- state transitions
- acceptance checking
- automata simulation

Concepts:
- automata theory
- formal languages
- stack-based computation

---

### ATAXX Game Logic
Implementation of the board game ATAXX logic.

Features:
- board representation
- move validation
- cloning and jumping mechanics
- state updates

Concepts:
- game logic
- grid algorithms
- state management

---

### Web Crawler
A simple crawler that visits pages and extracts content.

Features:
- URL traversal
- basic parsing
- recursive crawling

Concepts:
- HTTP requests
- parsing
- traversal algorithms

---

### To-Do Application
A small task-management project.

Features:
- add tasks
- remove tasks
- mark tasks as completed

Concepts:
- CRUD operations
- state management
- simple application design

---

### Essentia Audio Experiments
Audio-analysis experiments using computational music-information-retrieval concepts.

Features:

- audio processing workflows

Concepts:
- signal processing
- MIR pipelines
- feature extraction
